## Development

### Prerequisites

- [Node.js](https://nodejs.org/) 20 or higher
- [NPM](https://www.npmjs.com/package/npm)

### Project setup

```sh
# Install NPM dependencies
$ npm install
```

### Run server

```sh
# Start vite development server
$ npm run dev
```

The webapp is now accessible at [http://localhost:8080](http://localhost:8080)

### Testing

```sh
# Run automated tests
$ npm test
```
